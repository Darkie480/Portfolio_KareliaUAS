# Bachelor's Thesis - Cat Gestation Tracker

**Course:** Bachelor's Thesis *(Work in Progress)*  
**Year:** 2026 
**Technologies used:** C#, .NET MAUI, XAML, MVVM, JSON 
**Tools used:** VisualStudio 2022, Git 
**Language:** Finnish  

## Project description

This repository contains the source-code and views for my Bachelor's thesis project. Documentation will be uploaded once the thesis has been completed and reviewed.
The goal of this project is to develop a cross-platform mobile application for pet owners (cat owners to start with) to allow them to follow their pets pregnancy, calculate the due date and follow the development process of the kittens throughout the gestation period.

---

## Features

- Add an animal to the application
- Calculate the expected due date
- Development information
- Educational tips for each stage
- Local data storage
- Modern MVVM architecture

## Learning objectives

This project focuses on:
- C# application development
- .NET MAUI cross-platform development
- MVVM architecture
- Mobile UI design with XAML
